# 🚗 Electric Vehicle Analysis Dashboard | Power BI

## 📌 Project Overview

The **Electric Vehicle Analysis Dashboard** is an interactive Power BI dashboard developed to analyze electric vehicle (EV) adoption, market trends, manufacturer performance, and geographical distribution. The dashboard transforms raw EV data into meaningful business insights through interactive visualizations, KPIs, and filters, enabling users to make informed decisions based on vehicle registration trends and market performance.

---

# 🎯 Project Objectives

- Analyze the growth of Electric Vehicles over different model years.
- Compare Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs).
- Identify the leading EV manufacturers and vehicle models.
- Visualize EV distribution across different states.
- Analyze CAFV (Clean Alternative Fuel Vehicle) eligibility.
- Develop an interactive dashboard for business intelligence and decision-making.
